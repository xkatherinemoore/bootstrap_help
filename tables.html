Bootstrap Tables

Tables in Bootstrap offer nice formatting with very little effort from a design perspective. The table below provides descriptions for the most useful classes.

BS Table Classes

Class Description
.table Applied to the table element; it provides standard formatting for the table
.table-color-* Can be applied to any table element; it will change the background color of that element to the color specified
.table-striped Applied to the table element; it creates striped rows
.table-striped-columns Applied to the table element; it creates striped columns
.table-hover Applied to the table element; it highlights a row of the table when the user hovers over it with the mouse (requires tbody element)
.table-active Applied to a tr, th, or td element; it highlights that element with the same formatting as if hovered
.table-bordered Applied to the table element; adds collapsed borders to the table (borders around each cell)
.table-borderless Applied to the table element; removes all borders from the table
.table-sm Removes padding to create a more consolidated table
.table-group-divider Can only be used in thead, tbody, or tfoot elements; creates a heavier top-border for the class element to separate sections
.table-responsive-* Applied to a div that wraps the table; it can be used to create a table that scrolls horizontally (breakpoints are optional)
