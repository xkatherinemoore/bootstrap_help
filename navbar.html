Bootstrap Navbar

Bootstrap's Navbar offers great functionality but can also be one of the more confusing aspects of the Bootstrap Framework. This page will only break down the basic structure of the navbar and explain the collapse functionality.

For a complete guide to all Bootstrap navbar features, visit the Bootstrap website.
W3Schools also has an easy to follow guide on some of the navbar features.

Creating a Navbar

Basic Steps:

  1. Give the nav element the classes .navbar and (usually) .navbar-expand-desired_breakpoint
  2. Include a div container that is the same width as the main content. This allows everything to line up. Another option is to use the .container-fluid class where the navbar will fill the screen
  3. Add the .navbar-brand class to any logos (Can be used for text, links, or photos)
  4. Create an unordered list (ul) with the class .navbar-nav
  5. Add list items (li) with the class .nav-item
  6. Add anchor elements (a) with the class .nav-link
  7. Add the .active class and aria-current="page" to the navigation link for the current page

Sample Code:


<nav class="navbar navbar-expand-md">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">MyLogo</a>
        <ul class="navbar-nav">
            <li class="nav-item">
                <a class="nav-link active" aria-current="page">Link 1</a>
            </li>
            <li class="nav-item">
                <a class="nav-link">Link 2</a>
            </li>
            <li class="nav-item">
                <a class="nav-link">Link 3</a>
            </li>
        </ul>
    </div>
</nav>

Toggling Navigation:

Bootstrap has a standard way of collapsing the navigation bar which makes it really useful for responsive websites, particularly in mobile

  • Note that at the breakpoint, the navbar links switch from an inline horizontal display to vertical block display
  • Toggling navigation will hide (collapse) the navigation menu/links until a button is pressed
  • While mobile-friendly navigation is the most common use of the .collapse class, it's worth noting that it can be used for menus outside of mobile

Basic Steps:

  1. Create a toggle button.
    • Class is .navbar-toggler
    • data-bs-toggle attribute adds Bootstrap's JavaScript toggle function
    • data-bs-target attribute is used to specify the target (this points to the id attribute of the collapsed div)
    • The aria-* attributes are used by screen readers and are covered more indepth here.
  2. Use span to add the toggler icon (class .navbar-toggler-icon) as the text for the button
  3. Add a div around the navigation menu
    • This will usually be around the ul with class .navbar-nav
    • Set div class to .collapse and .navbar-collapse
    • Set a relevent id for the div. This should be the data-bs-target value for the .navbar-toggler

Sample Code:


<button class="navbar-toggler" type="button" 
data-bs-toggle="collapse" data-bs-target="#my-nav-id" 
aria-controls="#my-nav-id" aria-expanded="false" 
aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
</button>

<div class="collapse navbar-collapse" id="my-nav-id">
    <ul class="navbar-nav"...</ul>
</div>
