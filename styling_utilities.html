Bootstrap Styling Utilities

Bootstrap supports faster styling through use of their styling utilities. These are bootstrap classes that can be added to modify the styling of an element, including background and text color, spacing, borders, sizing, and text. This page provides only a brief overview of the functionality supplied by Bootstrap Utilities

Color

Color Classes

The classes below are the standard color classes in Bootstrap. They can be used to set the background or text color of any element by adding the appropriate class. Note that the .text-bg-* adjusts the background color to whatever is specified and then changes the font color to a high-contrast color. These classes can also be used to create themes which is covered here.

Spacing

Bootstrap spacing utilities includes classes to specify margins, padding, and gap. The classes range in size from 0 (no margin/padding/gap) - 5 (largest margin/padding/gap), as well as an auto function. They support breakpoints and have options to specify sides. The table below breaks down the possible class options.

Note: The container, row, and column classes for divs have a special property called gutters (g) which can also be used to adjust the padding and margins of these elements.

[property][side]-[breakpoint][size]

A property and size is required, but the side and breakpoint may be omitted.

Property Side Breakpoint Size
m (margin) blank (all) sm 0
p (padding) t (top) md 1
gap (gap) b (bottom) lg 2
s (start/left) xl 3
e (end/right) xxl 4
x (left & right) 5
y (top & bottom) auto
