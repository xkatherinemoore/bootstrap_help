Bootstrap Styling Utilities

Bootstrap supports faster styling through use of their styling utilities. These are bootstrap classes that can be added to modify the styling of an element, including background and text color, spacing, borders, sizing, and text. This page provides only a brief overview of the functionality supplied by Bootstrap Utilities

Color

Color Classes

The classes below are the standard color classes in Bootstrap. They can be used to set the background or text color of any element by adding the appropriate class. Note that the .text-bg-* adjusts the background color to whatever is specified and then changes the font color to a high-contrast color. These classes can also be used to create themes which is covered here.

Background (.bg-*) Text (.text-*) Text-Background (.text-bg-*)
.bg-primary .text-primary .text-bg-primary
.bg-secondary .text-secondary .text-bg-secondary
.bg-success .text-success .text-bg-success
.bg-danger .text-danger .text-bg-danger
.bg-warning .text-warning .text-bg-warning
.bg-info .text-info .text-bg-info
.bg-light .text-light .text-bg-light
.bg-dark .text-dark .text-bg-dark

Opacity

In addition to the standard colorings, Bootstrap also supports adjusting the opacity using classes.

Color-related Styling Classes

Category Class Description
Background .bg-* Adjusts background color
.bg-gradient Applies a linear gradient (as a background image) which fades to the bottom
text-bg-* Adjusts background color and applies contrasting font color
Text text-* Adjusts font color
link-* Adjusts the color of the link text while still maintaining link functionality (like hover)
Opacity opacity-* Adjusts the opacity of an entire object
bg-opacity-* Adjusts the background opacity
text-opacity-* Adjusts the text opacity
